Match Codes & Relays

If you happen to experience dodgy connections through Fightcade, or a lobby for your game is unavailable on the service, you can connect directly to your opponent within Flycast Dojo!

Using P2P Match Codes & Relay Servers, Flycast Dojo can start game sessions behind firewalls, so that you can play against others without the need for a Virtual LAN or matchmaking service.

Open Fightcade\emulator\flycast\flycast.exe and select MATCH CODE. A code will be given to exchange with your opponent.

You can also try out a relay server near you if Match Code hole punching doesn't work. Select RELAY instead and follow the same steps as Match Codes. Just add a relay server address near you and go.
